
# Breadcrumb

面包屑用于识别页面在层次结构内的位置，并根据需要向上返回。

import Breadcrumb,{BreadcrumbObjectsType} from '../../../src/components/Breadcrumb/breadcrumb';
import ShowBar from '/src/components/ShowBar';
import {objectsExample_1} from '/src/components/BreadcrumbExample';
import '../../../src/styles/doc.scss';

## 引入

```js
import Breadcrumb,{BreadcrumbObjectsType} from '@triangle-ui/breadcrumb';
```

## 使用示例

### 基本使用

通过创建一个对象数组来使用面包屑组件。  

```jsx
const objectsExample: BreadcrumbObjectsType[] = [
  {
    text: '首页',
    link: '/',
  },
  {
    text: '文档',
    link: '/docs/intro',
  },
  {
    text: '面包屑',
    link: '/docs/components/breadcrumb',
  },
];


export default function Example() => {
  return (
      <Breadcrumb objects={objectsExample} />
  );
};
```

<ShowBar>
    <Breadcrumb objects={objectsExample_1} />
</ShowBar>

### 跳转方式

通过设置 `target` 属性来设置跳转方式。有两个可选值： `_blank` 和 `_self` 。

```jsx
<Breadcrumb target={'_blank'} objects={[{text:'首页',link:'/'},{text:'文档',link:'/docs/intro'}]} />
```

<ShowBar>
          <Breadcrumb target={'_blank'} objects={[{text:'首页',link:'/'},{text:'文档',link:'/docs/intro'}]} />
</ShowBar>
