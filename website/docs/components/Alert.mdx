# Alert

import Alert,{AlertProps,AlertType} from '../../../src/components/Alert/alert';
import AlertShow from '/src/components/AlertShow';
import ShowBar from '/src/components/ShowBar';
import '../../../src/styles/doc.scss';

提示，展现需要关注的信息。  

## 引用

```js
import Alert,{AlertProps,AlertType} from '@triangle-ui/alert';
```

## 使用示例

### 默认

```jsx
<Alert title="默认提示" description="这是一条默认提示" />
```

<ShowBar>
  <Alert title="默认提示" description="这是一条默认提示" />
</ShowBar>

### 多种类别提示

```jsx
<Alert title="成功提示" description="这是一条成功提示" alertType={AlertType.Success} />
<Alert title="警告提示" description="这是一条警告提示" alertType={AlertType.Warning} />
<Alert title="危险提示" description="这是一条危险提示" alertType={AlertType.Danger} />
```

<ShowBar>
  <Alert title="成功提示" description="这是一条成功提示" alertType={AlertType.Success} />
  <Alert title="警告提示" description="这是一条警告提示" alertType={AlertType.Warning} />
  <Alert title="危险提示" description="这是一条危险提示" alertType={AlertType.Danger} />
</ShowBar>

### 带有关闭按钮

```jsx
<Alert title="默认提示" description="这是一条默认提示" closeable />
```

<ShowBar>
  <Alert title="默认提示" description="这是一条带有关闭按钮的默认提示" closeable />
</ShowBar>

可以通过 `onClose` 属性来监听关闭事件。

```jsx
<Alert title="警告提示" description="这是一条带有关闭按钮的警告提示" closeable alertType={AlertType.Warning} onClose={() => {alert('点击了关闭按钮')}} />
```

<ShowBar>
  <Alert title="警告提示" description="这是一条带有关闭按钮的警告提示" closeable alertType={AlertType.Warning} onClose={() => {alert('点击了关闭按钮')}} />
</ShowBar>

### 不同位置

```jsx
<Alert title="Left Alert" description="This is a left alert" alertType={AlertType.Success} position="left" />
<Alert title="Right Alert" description="This is a right alert" alertType={AlertType.Warning} position="right" />
```

<AlertShow/>

## API

| 参数 | 说明 | 类型 | 默认值 |
| :---: | :---: | :---: | :---: |
| className | 自定义类名 | string | - |
| alertType | 提示类型 | `AlertType.Default`  `AlertType.Success` <br/> `AlertType.Warning`  `AlertType.Danger` | `AlertType.Default` |
| title | 标题 | string | - |
| description | 描述 | string | - |
| closeable | 是否可关闭 | boolean | `false` |
| position | 位置 | 'left' \| 'right' | - |   
| onClose | 关闭时触发的回调函数 | `React.MouseEventHandler<HTMLButtonElement>` | - |
