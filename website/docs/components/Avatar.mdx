# Avatar

import Alert,{AlertProps,AlertType} from '../../../src/components/Alert/alert';
import Avatar,{AvatarSize,AvatarType} from '../../../src/components/Avatar/avatar';
import ShowBar from '/src/components/ShowBar';
import '../../../src/styles/doc.scss';

用作头像显示，可以为图片或字符形式展示。

## 引入

```js
import Avatar,{AvatarSize,AvatarType} from '@triangle-ui/avatar';
```

## 使用示例

### 默认使用

```jsx
<Avatar/>
```

<ShowBar>
    <Avatar/>
</ShowBar>

### 图片

通过设置 `src` 属性来设置图片。

```jsx
<Avatar src="https://avatars.githubusercontent.com/u/122715946?s=200&v=4"/>
<Avatar src="https://avatars.githubusercontent.com/u/83146544?v=4"/>
```

<ShowBar>
    <Avatar src="https://avatars.githubusercontent.com/u/122715946?s=200&v=4"/>
    <Avatar src="https://avatars.githubusercontent.com/u/83146544?v=4"/>
</ShowBar>

### 文字

通过设置 `text` 属性来设置文字，会根据文字长度自动调整字符显示数量。  

```jsx
<Avatar text="A"/>
<Avatar text="Smith"/>
```

<ShowBar>
    <Avatar text="A"/>
    <Avatar text="Smith"/>
</ShowBar>

<br/>

<Alert alertType={AlertType.Warning} title="注意" description="当文字和图像同时使用时，文字将不会显示。"/>


### 多种颜色

通过设置 `avatarType` 属性来设置颜色。可选值为 `primary`、`success`、`secondary`、`danger`、`warning` 。

```jsx
<Avatar text="A" avatarType={AvatarType.Primary}/>
<Avatar text="B" avatarType={AvatarType.Success}/>
<Avatar text="C" avatarType={AvatarType.Secondary}/>
<Avatar text="D" avatarType={AvatarType.Danger}/>
<Avatar text="E" avatarType={AvatarType.Warning}/>
```

<ShowBar>
    <Avatar text="A" avatarType={AvatarType.Primary}/>
    <Avatar text="B" avatarType={AvatarType.Success}/>
    <Avatar text="C" avatarType={AvatarType.Secondary}/>
    <Avatar text="D" avatarType={AvatarType.Danger}/>
    <Avatar text="E" avatarType={AvatarType.Warning}/>
</ShowBar>


### 大小

通过设置 `size` 属性来设置大小。可选值为 `small`、`large` 。默认不填时，为适中大小。

```jsx
<Avatar text="A" size={AvatarSize.Small}/>
<Avatar text="B"/>
<Avatar text="C" size={AvatarSize.Large}/>
```

<ShowBar>
    <Avatar text="A" size={AvatarSize.Small}/>
    <Avatar text="B"/>
    <Avatar text="C" size={AvatarSize.Large}/>
</ShowBar>

### 形状

通过设置 `circle` 属性来设置头像为圆形。默认不填时，为方形。

```jsx
<Avatar text="A" circle/>
```

<ShowBar>
    <Avatar text="A" circle/>
</ShowBar>

形状可与上述属性叠加使用。

```jsx
<Avatar text="A" circle size={AvatarSize.Large}/>
<Avatar text="B" circle size={AvatarSize.small} avatarType={AvatarType.Secondary}/>
```

<ShowBar>
    <Avatar text="A" circle size={AvatarSize.Large}/>
    <Avatar text="B" circle size={AvatarSize.small} avatarType={AvatarType.Secondary}/>
</ShowBar>

## API


| 参数 | 说明 | 类型 | 默认值 |
| :---: | :---: | :---: | :---: |
| className | 自定义类名 | string | - |
| avatarType | 颜色 | `AvatarType.Default` `AvatarType.Primary` `AvatarType.Secondary` `AvatarType.Success` `AvatarType.Danger` `AvatarType.Warning` | `AvatarType.Default` |
| size | 大小 | `AvatarSize.Small` `AvatarSize.Large` | - |
| src | 图片地址 | string | - |
| alt | 图片描述 | string | - |
| circle | 是否为圆形 | boolean | false |
| text | 文字 | string | - |
