
# Button

import Button, { ButtonSize, ButtonType } from '../../../src/components/Button/button';
import ShowBar from '/src/components/ShowBar';
import '../../../src/styles/doc.scss';

按钮允许用户执行操作并通过单击进行选择。

## 引用

```jsx
import Button, { ButtonSize, ButtonType } from '@triangle-ui/button';
```


## 使用示例

### 默认

```jsx
<Button>Default</Button>
```
<ShowBar>
    <Button>Default</Button>
</ShowBar>

### 多种颜色

您可以使用颜色属性更改按钮的颜色。  

```jsx
<Button btnType={ButtonType.Primary}>Primary</Button>
<Button btnType={ButtonType.Secondary}>Secondary</Button>
<Button btnType={ButtonType.Success}>Success</Button>
<Button btnType={ButtonType.Danger}>Danger</Button>
```
<ShowBar>
    <Button btnType={ButtonType.Primary}>Primary</Button>
    <Button btnType={ButtonType.Secondary}>Secondary</Button>
    <Button btnType={ButtonType.Success}>Success</Button>
    <Button btnType={ButtonType.Danger}>Danger</Button>
</ShowBar>

### 描边

```jsx
<Button btnType={ButtonType.Outline}>Outline</Button>
```
<ShowBar>
    <Button btnType={ButtonType.Outline}>Outline</Button>
</ShowBar>

### 链接

链接是一种特殊的按钮，它将用户带到另一个页面。  

```jsx
<Button btnType={ButtonType.Link} href="http://www.bing.com">Bing Link</Button>
<Button btnType={ButtonType.Link} target={'_blank'} href="http://www.bing.com">Bing Link</Button>
```

<ShowBar>
    <Button btnType={ButtonType.Link} href="http://www.bing.com">Bing Link</Button>
    <Button btnType={ButtonType.Link} target={'_blank'} href="http://www.bing.com">Bing Link</Button>
</ShowBar>

### 禁用

无法使用和无法点击的按钮。以上所有按钮都可以设置为禁用状态。  

```jsx
<Button disabled>Disabled</Button>
<Button btnType={ButtonType.Primary} disabled>Primary</Button>
<Button btnType={ButtonType.Secondary} disabled>Secondary</Button>
<Button btnType={ButtonType.Success} disabled>Success</Button>
<Button btnType={ButtonType.Danger} disabled>Danger</Button>
<Button btnType={ButtonType.Link} href="http://www.bing.com" disabled>Bing Link</Button>
<Button btnType={ButtonType.Outline} disabled>Outline</Button>
```

<ShowBar>

## API


| 参数 | 说明 | 可用值 | 默认值 |
| :---: | :----: | :---: | :---: |
| btnType | 按钮类型 | `ButtonType.Primary` <br/> `ButtonType.Danger`  `ButtonType.Secondary` <br/> `ButtonType.Success`  `ButtonType.Link`  `ButtonType.Outline` | `ButtonType.Default` |
| size | 按钮大小 | `ButtonSize.Large` <br/> `ButtonSize.Small` | `ButtonSize.Default` |
| disabled | 是否禁用 | `boolean` | `false` |
| href | 链接地址 | `string` | - |
| children | 按钮内容 | `ReactNode` | - |
| className | 自定义类名 | `string` | - |
| style | 自定义样式 | `CSSProperties` | - |
| onClick | 点击事件 | `React.MouseEventHandler` | - |

以及所有原生的 `button` 元素的属性。当使用 `btnType` 为 `ButtonType.Link` 时，可以使用所有 `a` 元素的属性。如 `target` `rel` 等。
